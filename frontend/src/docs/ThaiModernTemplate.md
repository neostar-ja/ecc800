# ECC800 เทมเพลตใหม่: คู่มือการใช้งาน

## ภาพรวม
เทมเพลตใหม่ของระบบ ECC800 ได้รับการออกแบบมาเพื่อปรับปรุงประสบการณ์ผู้ใช้ (UX) และการมองเห็น (UI) โดยใช้ความสามารถของ Material UI และ Tailwind CSS ร่วมกัน เทมเพลตนี้มุ่งเน้นที่การใช้งานภาษาไทย การออกแบบที่ทันสมัย และการแสดงผลที่ปรับเปลี่ยนตามขนาดหน้าจอ (Responsive Design)

## คุณสมบัติหลัก
1. **เลย์เอาท์แบบทันสมัย**: ใช้เลย์เอาท์ที่สะอาดและทันสมัยพร้อมการใช้พื้นที่หน้าจออย่างมีประสิทธิภาพ
2. **การรองรับภาษาไทย**: ทุกองค์ประกอบและป้ายกำกับใช้ภาษาไทยเพื่อการใช้งานที่สะดวกยิ่งขึ้น
3. **โหมดกลางวัน/กลางคืน**: การสลับระหว่างธีมแบบสว่างและมืดที่ราบรื่น
4. **การแสดงผลแบบปรับตัว**: รองรับทุกขนาดหน้าจอจากมือถือถึงเดสก์ท็อป
5. **การแจ้งเตือนที่ทันสมัย**: ระบบการแจ้งเตือนที่ได้รับการปรับปรุงเพื่อการมองเห็นที่ดีขึ้น
6. **เมนูนำทางที่ใช้งานง่าย**: เมนูด้านข้างที่จัดระเบียบพร้อมไอคอนและคำอธิบาย

## วิธีการใช้งาน

### เมนูนำทาง
เมนูนำทางอยู่ทางด้านซ้ายของหน้าจอและประกอบด้วยรายการต่อไปนี้:
- **แดชบอร์ด**: ภาพรวมและการตรวจสอบ
- **ไซต์**: การจัดการไซต์
- **อุปกรณ์**: สถานะฮาร์ดแวร์
- **เมทริกซ์**: ข้อมูลประสิทธิภาพ
- **ข้อผิดพลาด**: การจัดการการแจ้งเตือน
- **รายงาน**: การวิเคราะห์และรายงาน
- **ผู้ดูแลระบบ**: การจัดการระบบ (สำหรับผู้ดูแลระบบเท่านั้น)

### แถบเครื่องมือด้านบน
แถบเครื่องมือด้านบนแสดงข้อมูลต่อไปนี้:
- **ชื่อหน้า**: แสดงชื่อหน้าปัจจุบันและคำอธิบายสั้นๆ
- **การแจ้งเตือน**: ไอคอนที่แสดงการแจ้งเตือนใหม่
- **โปรไฟล์ผู้ใช้**: เมนูแบบเลื่อนลงสำหรับตัวเลือกผู้ใช้ เช่น โปรไฟล์, ตั้งค่า, และออกจากระบบ

### ส่วนท้าย
ส่วนท้ายแสดงข้อมูลต่อไปนี้:
- **ลิขสิทธิ์**: ข้อมูลลิขสิทธิ์
- **เวอร์ชัน**: เวอร์ชันปัจจุบันของระบบ
- **ปุ่มช่วยเหลือ**: ปุ่มสำหรับความช่วยเหลือ, ข้อมูล, และการเปลี่ยนภาษา
- **ข้อมูลผู้พัฒนา**: แสดงข้อมูลผู้พัฒนาระบบ

### การสลับโหมดกลางวัน/กลางคืน
คุณสามารถสลับระหว่างโหมดกลางวันและกลางคืนได้โดยใช้ปุ่มสลับในเมนูด้านข้าง ระบบจะจำการตั้งค่าของคุณสำหรับการเข้าชมในครั้งต่อไป

### การแสดงผลบนอุปกรณ์มือถือ
บนอุปกรณ์มือถือและแท็บเล็ต เมนูด้านข้างจะถูกซ่อนเริ่มต้นและสามารถเข้าถึงได้ผ่านปุ่มเมนูในแถบเครื่องมือด้านบน องค์ประกอบอื่นๆ ทั้งหมดจะปรับขนาดและจัดเรียงใหม่เพื่อการใช้งานที่ดีที่สุดบนขนาดหน้าจอที่เล็กลง

## การปรับแต่ง
หากต้องการปรับแต่งเทมเพลตเพิ่มเติม คุณสามารถแก้ไขไฟล์ `ThaiModernLayout.tsx` ได้ ไฟล์นี้มีตัวแปรสำหรับสีและขนาดองค์ประกอบต่างๆ ที่สามารถปรับแต่งได้

## การแก้ไขปัญหา
- **เมนูนำทางไม่แสดง**: ตรวจสอบว่าคุณเข้าสู่ระบบถูกต้องและมีสิทธิ์การเข้าถึงที่เหมาะสม
- **โหมดกลางคืนไม่ทำงาน**: ตรวจสอบว่าเบราว์เซอร์ของคุณรองรับคุกกี้และ localStorage
- **ปัญหาการแสดงผล**: ลองรีเฟรชหน้าจอหรือล้างแคชของเบราว์เซอร์

## ข้อมูลเทคนิค
- **Framework**: React + TypeScript
- **UI Libraries**: Material UI 5.14.15 + Tailwind CSS 3.3.5
- **State Management**: Zustand (สำหรับ Authentication)
- **Routing**: React Router 6.8
- **Theme**: ใช้ React Context สำหรับการจัดการธีม
